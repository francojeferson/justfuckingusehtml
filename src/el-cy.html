Γιατί το HTML εν πελλαμός

Ούλλοι ξέρουν HTML

Άκου ρε βλάκα. Ού-λλοι-στο-θκιά-ολο ξέρουν HTML. Η στετέ σου; Ήταν μέστα χαντάτζια στον δεύτερο παγκόσμιο τζαι επολεμούσε τους ναζί την ίδια ώρα που έγραφε <table> layouts μέστο δευτερούι της για να στείλει κρυπτογραφημένα μηνύματα στους συμμάχους. Ο παππούς σου; Ο πελλός έσσει ολόκληρο PhD παστο HTML που το Πανεπιστήμιο Σταρ-Τζιθκια-Μας, τζαι έσσει το να ρέμπει δίπλα που το βραβείο του "Επέζησα το dial-up". Ο τζίρης σου; Ελαλούσε <div> τζαι <span> πριν καν πει μανά. Θκιάολε μαύρε, ακόμα τζαι ο ψύρης ο σσίλλος σου πέρκει να πουλά HTML σελίδες πάστο Fiverr. Οπόταν γιατί στον πούτσο καθούμαστε τζαι απαντούμε στες μαννοερωτήσεις σου; Κύριε ελέησον πκιόν.

Κανένα γαμημένο hydration error. Πκιέ λλίο νερό, μαλάκα

Τι στον γαμώσσιστο εν το hydration error; Ακούετε σαν μια hipster μαλακία που εσπούρτησε κάποιος baristas επειδή το γαμημένο σου το latte με oat milk, μίσσιμου, εν εμούσσεψε καλά. "Sorry κουμπάρε, ο καφές σου έσσει hydration error". Τί στον πούτσο! Τζαι το "tree shaking"; Σοβαρομιλούμε τωρά; Που είμαστε ρε θκιάολε; Σε κατασκήνωση για προγραμματιστές όξα σε συνέδριο ξυλοκόπων; "Ναί μάστρε, έφαα ούλλην την μέρα μου να σσουμαλίζω δέντρα πέρκι λλιάνει το JavaScript. Πούντο βραβείο μου;" Ποιός ξηππασμένος μαλάκας ανακάλυψε τούντες μαλακίες; Μαλάκα μου, εν έχω καθόλου ώρα να αποκρυπρογραφήσω τούντες πελλάρες ούλλες. Το HTML εν έσσει έτσι ιστορίες. Απλά δουλεύκει.

Το HTML εν χρειάζεται "υποστήριξη"

Είδες ποττέ κανένα μαλάκα να προσλαμβάνει άτομα για να γλέπουν το HTML; Ακριβώς, κανένας εν κάμνει έτσι μαλακίες! Το HTML εν νώθει. Εν θέλει 24ωρη γραμμή βοήθειας σαν κάποια μαμμόθρευτα frameworks. Την ώρα που τα γαμωframeworks σου θωρούν περίοδο κάθε φορά που κάμνει update ο browser σου τζαι κλέουντε για patches κάθε κάρτον, το HTML γύρνει πίσω, τσακρά μιαν παωμένην τζαι κάμνει τι στον πούτσο πρέπει να κάμει. Γιαυτό, την επόμενη φορά που κάποιος κλαρινογαμπρός πωλητής προσπαθήσει να σου μπήξει "premium HTML support", να του πεις να πάει να γαμηθεί.

Το HTML εν χρειάζεται "deploy"

Να κάμεις deploy HTML; Πολλά πιο εύκολο που το να σβήσεις τες πορνοσελίδες που τον browser σου μετά που μαραθώνιο μουτσιάς. Απλά βάλε τα file σου πάσσε έναν web server τζαι τέλος. Ούλλο το γαμημένο ίντερνετ μπόρει να δει την μαλακία που έκαμες. Χωρίς γαμημένα deployment process, χωρίς γαμημένα DevOps, χωρίς γαμημένα Docker container ή CI/CD μαλακίες. Απλά κάμνεις drag, drop τζαι κανεί. Εντωμεταξύ τα γαμωframeworks σου θέλουν την ίδια διαδικάσία που θέλει η ΝΑSA να εκτοξεύσει πύραυλο απλά για να βάλουν πάνω ένα κουμπούι που λαλεί "Hello, World". Σταρτζίθκια του HTML όμως για τες μαλακίες που χρησειμοποιάς για deployment. Εν όπως τον καφενέ του χωρκού στο ιντερνετ: πάντα ανοικτό, χωρίς πολλά πολλά, απλά δουλεύκει. Το γαμωframework σου; Το hipster κκαφέ που εν "προσωρινά κλειστό" κάθε φορά που το χρειάζεσαι.

Μαλάκα, εν θέλεις πατρονιές UI για να κάμεις κάτι να φαίνεται καλό

Το μόνο που χρειάζεσαι εν τον γαμημένον τον νούν σου. Κάθε χρόνο κυκλοφορά ακόμα μια φρέσκα σκατίλλα: γαμωframeworks που δουλεύκκουν ττάππο τζαι εν θκυό κλαννιές πιο καλά που τα προηγούμενα "επαναστατικά" σκατά. Κάποιοι λαλούν εν πρόοδος. Ακόμα πιο στροντζιλές γωννιούες, καινούργια χρώματα τζαι ακόμα ένα πράμα που πρέπει να μάθεις.

Το HTML βάλλει πατερίτσες στο γαμημένο JS σου

Ναι είσαι, γαμημένε.

Ήξερες πως άμαν βάλεις ένα id σε ένα HTML element, το HTML εν κάθεται απλά τζαμέ σαν τον κουνόσσιλλο; Που τ'αλήθκεια κάμνει σου μια γαμημένη μεταβλητή μέστο JavaScript. Ναι, καλά με άκουσες ρε μαλάκα. Εν χρειάζεται καν να σηκώσεις το δαχτυλούι σου. Το HTML κάμνει τη σοβαρή τη δουλειά ενώ το JS σου ακόμα προσπαθεί να μάθει πώς να δίννει τα κορδόνια του.

Δοκίμασε να δείς, άνοιξε το console τζαι γράψε "ειμαι_μαλακας". Άδε! Έτο τζαμέ ρε σκατούι. Το HTML μόλις σου έδωκε μια μεταβλητή μέστον δίσκο, τζαι εν χρειάστηκε καν να κλαφτείς. Τούτα εν βασικά πράματα, τζαι αν μεν το ήξερες τούτο, μπράβο σου. Είσαι επίσημα πιο παλαβός τζαι που μια σακούλα ρότσους.

Τούτο εν το HTML τζαι δείχνει σου τη δύναμή του, τζαι πως έννεν απλά μια στατική σσύλλα. Εν δυναμικό, έσσει δύναμη τζαι κουβαλάσε που την πρώτη μέρα σαν τον καημένο. Εντωμεταξύ το JS σου γυρίζει τζαι μάσσετε παντές εν ο σούπερσταρ του show ενώ στην πραγματικότητα απλά εν πάστες ράσσιες του HTML.

Οπόταν την επόμενη φορά που εννα τον παίζεις με τα γαμωframeworks σου να θυμάσαι: Το HTML εν το μόνο που κάμνει την σοβαρή δουλειά τζαι περιπαίζει τες μαλακίες που κάμνεις.
